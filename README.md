# 🚀 Witaj na mrfk1411.github.io!

Siema! To jest repozytorium mojej osobistej strony internetowej, hostowanej bezpośrednio przez **GitHub Pages**. Projekt służy jako moja wizytówka w sieci, miejsce do testowania nowych rozwiązań frontendowych oraz hub dla moich mniejszych projektów.

---

## 🛠️ Tech Stack & Narzędzia

Do stworzenia i utrzymania tej strony wykorzystuję:

*   **Frontend:** HTML5, CSS3 (z wykorzystaniem nowoczesnych animacji i Flexbox/Grid), JavaScript (ES6+)
*   **Hosting & Deployment:** GitHub Pages / GitHub Actions
*   **Środowisko:** Visual Studio Code & Git

---

## 🌟 Funkcje strony

*   📱 **Pełna responsywność (RWD):** Strona wygląda świetnie zarówno na potężnym monitorze, jak i na ekranie smartfona.
*   🎨 **Modern Design:** Czysty, nowoczesny interfejs z intuicyjną nawigacją.
*   ⚡ **Wydajność:** Szybkie ładowanie dzięki optymalizacji kodu i zasobów.
*   🌓 **Dark Mode (Opcjonalnie):** Wbudowany przełącznik motywu dla nocnych marków.
